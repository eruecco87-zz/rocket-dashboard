Box Container type

This is the main container type, it has a header color that can be defined with an extra box class: .box-info, .box-danger, .box-primary, .box-success, .box-warning or .box-clear, if no color class is provided it will default to a grey color.

A box can have optional header buttons or tabs and footer.
