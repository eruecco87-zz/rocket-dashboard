Javascript

Tables are automatically initialized using the same code as described here, but you can initialize them using your own selectors with the javascript utility:

rocketDashboard.Utils.tableCollapse(
  'table-selector',
  'trigger-selector',
  'collapsed-selector',
  'closed-icon-seletor',
  'open-icon-selector'
);

Code

To create a collapsible table, every second row will need a class .row-collapsed and will act as it's previous row's description, it will also need a colspan="*" equal to the table's total columns.

On the main data row, a class of .row-collapsed-trigger will indicate the element that triggers the collapsed row display. Optionaly you can insert a font awesome icon <i class="fa fa-plus-square"></i> before the text to act as state indicators

Every collapsed row can also have a contextual class for success, error, info, etc. .row-success, .row-danger, .row-warning,
.row-info, .row-default, .row-primary
