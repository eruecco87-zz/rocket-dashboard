General theme layout

  1. UI Elements
  2. General

Main Layout

The main layout is constituted by the Top Nav, the Drawer Menu and the Page Container; this last one contains the bootstrap columns that hold the Main Container and the Sidebar.

Alternatively a Bottom Nav can be added to the layout just after the Top Nav

As a starting point you can use the blank page bundled with the rocket dashboard download. Here's an example of how a basic layout would look:

View HTML

The sidebar column can be added before the page content column to place the sidebar to the left.

To have a full width layout without a sidebar, make the page content column 12 columns wide and take out the .has-sidebar class from the page container.

Top Nav

The top navigation contains the drawer menu trigger and 3 different types of dropdowns: Messages, Notifications and Tasks.

Alternatively you can add a .top-nav-fixed class to attach the top nav to the screen and have it scroll along with the user.

Bottom Nav

The bottom navigation is a regular bootstrap navigation, documentation for bootstrap's navbar can be found here.

Alternatively you can add a .bottom-nav-fixed class to attach the bottom nav to the screeen and have it scroll along with the user.

Drawer Menu

The drawer menu is a Jasny Bootstrap navmenu component. It can have a .navmenu-fixed-right class or a .navmenu-fixed-left class to place it to either side. To close it simply click anywhere in the UI outside the drawer menu.

Sidebar

The sidebar can have an alternative .sidebar-fixed class to attach it to the screen and have it scroll along with the user.

If the sidebar content is lager than the viewport a scrollbar will be added so the user can get to the bottom content.

You can add a div with a class of .sidebar-section to create a bottom bordered section separator such as this one.
