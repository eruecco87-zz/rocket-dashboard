## Rocket Dashboard (Admin Template)
### [Demo & Docs](http://eruecco87.github.io/rocket-dashboard/)
----------
[Bootstrap](http://getbootstrap.com/) based admin template built with [yeoman](http://yeoman.io/) and packed with the following javascript and css goodies:

* [jQuery](http://jquery.com/)
* [jQueryUI](http://jqueryui.com/)  
* [Bootstrap-Jasny](http://jasny.github.io/bootstrap/)
* [Font-Awesome](http://fortawesome.github.io/Font-Awesome/icons/)
* [Ionicons](http://ionicons.com/)
* [Pace](http://github.hubspot.com/pace/docs/welcome/)
* [Spin.js](http://fgnass.github.io/spin.js/)
* [Messenger](http://github.hubspot.com/messenger/)
* [jvectormap](http://jvectormap.com/)
* [DynaTable](http://www.dynatable.com/)
* [Morris.js](http://morrisjs.github.io/morris.js/)
* [jQuery Slismscroll](http://rocha.la/jQuery-slimScroll)
* [jQuery Knob](https://github.com/aterrien/jQuery-Knob)
* [jQuery Peity](http://benpickles.github.io/peity/)
* [x-editable](http://vitalets.github.io/x-editable/)
* [MomentJS](http://momentjs.com/)
* [Moment Range](http://gf3.github.io/moment-range/)
* [Bootstrap Datepicker](http://eternicode.github.io/bootstrap-datepicker/)
* [ion.RangeSlider](http://ionden.com/a/plugins/ion.rangeSlider/en.html)
* [Jquery Minicolors](http://labs.abeautifulsite.net/jquery-minicolors/)
* [Bootstrap Touchspin](http://www.virtuosoft.eu/code/bootstrap-touchspin/)
* [Bootstrap Switch](https://github.com/nostalgiaz/bootstrap-switch/)
* [jQuery iCheck](http://fronteed.com/iCheck/)
* [Bootstrap Select](http://silviomoreto.github.io/bootstrap-select/3/)
* [BootstrapValidator](http://bootstrapvalidator.com/)
* [Ladda Bootstrap](http://msurguy.github.io/ladda-bootstrap/)
* [bootstrap3-wysihtml5](https://github.com/Waxolunist/bootstrap3-wysihtml5-bower/)
